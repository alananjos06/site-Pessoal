# Dev Girls
Website de uma equipe de quatro desenvolvedoras, apresentando perfis individuais, portfólio de projetos, novidades e contato.

## 👩‍💻 Equipe:
- **Kelly** — CEO & Fundadora
- **Giovanna** — Gerente de Projetos
- **Alana** — Full Stack Developer
- **Monique** — Back End Developer

## Páginas Feitas:
- **index.html** — Página Inicial
- **sobre-kelly.html** — Perfil da Kelly
- **sobre-giovanna.html** — Perfil da Giovanna
- **sobre-alana.html** — Perfil da Alana
- **sobre-monique.html** — Perfil da Monique
- **portfolio.html** — Portfólio
- **novidades.html** — Novidades
- **contato.html** — Contato

## 🛠️ Tecnologias Utilizadas:
- HTML5
- CSS3
- JavaScript
- Font Awesome (ícones)
- Google Fonts (fonte Poppins)

## Funcionalidades:
- Menu vertical lateral navegável com submenu dropdown
- Menu horizontal alternativo (comentado no arquivo, caso queira trocar manualmente para testar)
- Páginas individuais de cada integrante com foto, habilidades, formação e experiência
- Portfólio com filtro por integrante feito em CSS
- Formulário de contato com validação e confirmação em JavaScript
- Tabela de redes sociais com links para LinkedIn, GitHub e Instagram de cada integrante
- Design responsivo para mobile e desktop

## ⚠️ Observações!
1 - Informações das integrantes são fictícias pois foi um trabalho acadêmico, realizado no intuito de mostrarmos nosso conhecimento em desenvolvimento web.<br>
2 - E ao contrário do site empresarial (Sabor&Brasa), este site é **100% estático** — feito apenas com HTML, CSS e JavaScript — e não precisa de servidor ou banco de dados para funcionar.
